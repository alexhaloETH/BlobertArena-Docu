Detailed Description

This is the lobby page, the user is only allowed here if the invite he sent or received has been accepted.

this is where the users now confront each other before the real battle starts or the countdown finishes

This page has two modes, one for the admin of the game and on for the guest of the game, depending on which we show different buttons

on enable the code should check if the game is in the correct state otherwise kick him back to the corresponding page, either the game page or main menu page
