◆ ButtonBehavior

enum StandardButton.ButtonBehavior

Defines the behavior of the button upon interaction.

  • ResetAfterDelay: Resets the button's visual state after a specified delay.
  • InstantChange: Changes the state instantly with no delay or locking.
  • AwaitCallback: Locks the button until explicitly unlocked via a callback.
