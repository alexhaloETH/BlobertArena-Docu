Detailed Description

Simple speach bubble script. After instantiation, the bubble is "dormant", to start it, call either the OneShotInsult() or ContinousInsulting() functions. To configure the lifetime and delay, use the SetLifetimeAndDelay() function before calling calling any of the starting functions. To stop the insults, call ResetBubble().
